# Unit 0 — Course Intro

A short (~15–20 min) orientation, delivered **once before Unit 1**. It sets up
how the whole course works:

- **Why this course exists** — graph + geo + AI, and why your edge is *directing*
  and *interpreting*, not typing.
- **The North Star** — DIRECT → CHOOSE → INTERPRET.
- **The weekly loop + rubric** — direct → interpret → extend (see
  [`../rubric.md`](../rubric.md)) and the
  [`decision log`](../decision-log-template.md).
- **The five-unit capability arc** and the bus-archive thread — see
  [`../SYLLABUS.md`](../SYLLABUS.md).
- **Onboarding** — fork, install, `uv sync --extra unit-1`, smoke test.

## Materials

- `theory.pdf` / `theory.html` — the orientation deck.

There is no demo, practice, or homework for this unit — start the hands-on work
in **Unit 1**.
